# BMI-Calculator-IOS
A simple BMI Calculator for your needs. Provide categorization.

## Status
| BMI            | Weight Status            |
|----------------|--------------------------|
| Below 18.5     | Underweight              |
| 18.5 – 24.9    | Normal or Healthy Weight |
| 25.0 – 29.9    | Overweight               |
| 30.0 and Above | Obese                    |
